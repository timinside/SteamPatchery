# SteamPatchery

**SteamPatchery** is a specialized patching utility designed to modify and enhance the **Steam Client**. It allows users to apply UI tweaks, re-enable hidden features, bypass client-side restrictions, and customize the overall Steam desktop experience.

## 🚀 Key Capabilities
* **Client Modification:** Apply patches directly to Steam's core files and resources.
* **UI/UX Enhancement:** Inject custom styles or restore legacy interface elements.
* **Feature Unlocking:** Toggle experimental or hidden Steam features.
* **Automated Backups:** Safely backs up original client files before applying any changes.

## 📖 Usage
1. Open the application and scan for your Steam installation path.
2. Select the available client patches from the list.
3. Click **Patch Client**.
4. Restart Steam to see the changes.

**SteamPatchery** is not affiliated with Valve Corporation. Modifying Steam client files may technically violate the Steam Subscriber Agreement. While typically safe for UI tweaks, use this tool at your own risk. This tool is intended for educational and customization purposes only.
